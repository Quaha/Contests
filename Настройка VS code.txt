1) Скачать VS code: https://code.visualstudio.com/download

2) Скачать MSYS2 и открыть при запуске (либо MSYS2 UCRT64): https://www.msys2.org/
3) Ввести: pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain
4) Добавить C:\msys64\ucrt64\bin в PATH

5) Скачать расширение C/C++ for Visual Studio Code
6) Скачать расширение Code-Runner

7) >Выбрать конфигурацию IntelliSense -> gcc
8) В параметрах code-runner.executorMap в settings.json вместо "cpp: ..." вставить такую строку
        "cpp": "cd $dir && g++ -std=c++23 -g $fileName -o main.exe && .\\main.exe",

9) Перейти в "Запуск и отладка" -> "Создайте файл launch.json" -> GDB/... -> gdb/Запустить
10) Заменить строку "program": "${workspaceFolder}/main.exe",
11) Заменить строку "miDebuggerPath": "C:/msys64/ucrt64/bin/gdb.exe",
12) Code-Runner: Run In Terminal -> Поставить галочку
